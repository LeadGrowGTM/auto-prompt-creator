SECTION: "## Step 4: Check Halt Conditions"
Check these conditions in order (first match halts):
1. Success: vNNN.json aggregate.overall_score >= 0.92 — HALT (threshold-reached)
2. Max iterations: loop-state.json current_iteration >= 15 — HALT (max-iterations)
3. Convergence plateau: last 3 score deltas all < 0.02 — HALT (convergence-plateau)
   Delta = |score_history[N].overall - score_history[N-1].overall|
4. Overfitting: |train_score - validation_score| > 0.08 — HALT (overfitting)
5. Token budget: estimated_tokens from Step 1 > 800 — HALT (token-budget-exhausted)

On any halt: set loop-state.json halt_reason to the halt string. Report best_version (score_history entry with highest val). Stop.
On continue: proceed to Step 5.

SECTION: "## Step 5: Semantic Drift Check (every 3 iterations)"
- Only run when current_iteration is divisible by 3 (iterations 3, 6, 9, ...)
- Read scenario.json description field
- Read current prompt (prompts/vNNN.md)
- Assess: "Does this prompt still serve the same task? Rate 1-5."
- If rating < 3: add "semantic_drift_warning": true to loop-state.json. Do NOT halt — flag and continue.

SECTION: "## Step 6: Generate Mutation"
- Read .planning/phases/03-mutation-optimization-loop/prompts/mutation-instruction-prompt.md

- Execute the combined text as Opus (you, the current session)
- Output: complete new prompt text + mutations.log entry (hold both in memory)

1. Write new prompt to: scenarios/icp-classification/prompts/v[N+1].md (padded 3 digits, e.g. v002)
2. Append mutations.log entry from Step 6 to: scenarios/icp-classification/mutations.log
   Fill in the actual score from Step 2 (replace TBD values)
3. Update scenarios/icp-classification/evals/loop-state.json:
   - current_iteration += 1
   - score_history: append {version: "vNNN", overall: X, train: X, val: X, tokens: X, timestamp: ISO8601}
   - best_version: update if new val > best_validation_score
   - best_validation_score: update if new val is higher
   - consecutive_plateaus: if |new.overall - prev.overall| < 0.02, increment; else reset to 0
   - last_mutation_type: set to type used (additive | consolidation | subtractive)

        scenarios/icp-classification/evals/v[N+1].json \

        scenarios/icp-classification/evals/loop-state.json
git commit -m "opt(icp-classification): v[N+1] -- [mutation-type] -- [overall_score]"
