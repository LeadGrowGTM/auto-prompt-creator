After each iteration of the anneal loop (per Plan 03 SKILL.md), the executor will update this file:
- Increment `current_iteration`
- Append new entry to `score_history`
- Update `best_version` and `best_validation_score` if new val score exceeds current best
- Increment `consecutive_plateaus` if score delta < 0.02, else reset to 0
- Set `halt_reason` when a halt condition fires

This file is the single source of truth for loop control. The anneal loop reads it at the start of each iteration and writes it at the end.
