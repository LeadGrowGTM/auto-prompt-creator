  <action>
Create `scenarios/icp-classification/evals/graduate.py` -- a Python script that:

1. Accepts one argument: the scenario directory path (e.g., `scenarios/icp-classification`). Defaults to the parent of the script's directory if not provided.
2. Reads `{scenario_dir}/evals/loop-state.json` and `{scenario_dir}/scenario.json`.
3. Finds the best version from `loop-state.json` `best_version` field.
4. Reads the corresponding prompt file: `{scenario_dir}/prompts/{best_version}.md`.
5. Looks up the best version's entry in `score_history` to get `overall`, `train`, `val`, `tokens`.
6. Counts the number of entries in `score_history` as `iterations_run`.
7. Reads `scenario.json` to get `name` and the number of inputs (count files matching *.md in `{scenario_dir}/inputs/`).
8. Creates `library/` directory at the project root (two levels up from evals/) if it doesn't exist.
9. Writes `library/{scenario_name}.md` with YAML frontmatter delimiters (triple-dash lines) containing these fields:
   - scenario: {scenario_name}
   - graduated: {ISO 8601 date only, e.g. 2026-03-31}
   - accuracy.overall: {overall_score}
   - accuracy.train: {train_score}
   - accuracy.validation: {validation_score}
   - threshold: {config.accuracy_threshold from scenario.json}
   - iterations: {iterations_run}
   - best_version: {best_version}
   - target_model: {config.target_model from scenario.json}
   - test_set_size: {number of input files}
   - tokens: {token count from score_history entry}
   Then after the closing frontmatter delimiter, the full prompt text from the best version file.
10. Prints a summary to stdout: scenario name, best version, scores, output path.
11. If `halt_reason` is null in loop-state.json, print a WARNING that the loop hasn't halted yet but still graduate the current best (the user may want to graduate early).

Per GRAD-01: The script copies the winning prompt to library/ as a named, portable file.
Per GRAD-02: The frontmatter includes scenario name, accuracy score, iterations needed, test set size, date.

After creating the script, run it:
```bash
python scenarios/icp-classification/evals/graduate.py scenarios/icp-classification
```

This will produce `library/icp-classification.md` with v001 as the current best (even though the loop hasn't reached threshold -- the script works regardless of halt state).
